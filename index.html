CDL Hardware Repository Documentation
This is the top level documentation page for the CDL hardware repository, on GitHub.

Contents

The branch contains the external CDL module documentation at present. It is available as a PDF document and directly in HTML.

Documentation branch creation

This documentation is in the gh-pages branch of the git repository, and is automatically made available on GitHub. The gh-pages branch was created using:

git clone https://github.com/atthecodeface/bbc_cdl.git gh-pages
cd gh-pages
git checkout --orphan gh-pages
git rm -rf .
... create and edit index.html and base.css ...
git add index.html base.css
git commit -m "First pages commit" -a
git push origin gh-pages

Thereafter, the documentation has been added to the gh-pages branch.

Documentation generation

The bulk of the documentation is generated using doxygen, from documentation (and a few comments) in the CDL source files and comments in the C source code.

Doxygen does not treat CDL natively, so a CDL filter is used in the documentation creation process. This part-parses the CDL and generates a Doxygen-commmented C output, which Doxygen can generate LaTeX and HTML documentation from.

git clone https://github.com/atthecodeface/bbc_cdl.git
cd bbc_cdl
make clean
make documentation
open doc/cdl_internal_html/index.html
 ... or ...
open doc/cdl_internal_latex/refman.pdf
